Project Name: Medie App
Project Description:

Medie App is a comprehensive content management application designed to provide users with an intuitive platform for publishing and sharing content across various categories. The project aims to enhance user experience by facilitating easy access to information and effective content organization.

Key Features:

User Management: Enables users to create accounts, customize profiles, and assign different roles (admin, author, reader).
Content Publishing: Allows users to create, edit, and delete articles with support for multimedia such as images and videos.
Comment Management: Encourages interaction by allowing readers to comment on articles, fostering community engagement.
Content Categorization: Features an efficient categorization system that enables users to browse content by categories (e.g., Technology, Health, Travel).
Flexible Settings: Provides options to customize site settings, including language preferences and the number of articles displayed per page.
Technologies Used:

Programming Language: PHP
Database: MySQL
Framework: Laravel 
Frontend: HTML, CSS, JavaScript
Goals:

Medie App aims to create a digital community where individuals can exchange knowledge and ideas, contributing to enhanced learning and social interaction.
